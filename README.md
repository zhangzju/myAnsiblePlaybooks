# myAnsiblePlaybooks
我的ansible playbook备份

## ping.yml
全局Ping的任务
 
## 文档站点专用
### docUpdate.yml
针对ispproject的文档站点进行文件同步

## 服务器专用
### backupGitlabConfigure.yml
备份gitlab服务器的配置文件

### backupRedmine.yml
备份Redmine的数据库文件和附件

### backupJenkins.yml
使用ThinBackup来备份Jenkins文件

# 使用说明
这部分ansible playbook是适配于tpdocumentsite仓库进行协作的
